spark 单点安装 1. 下载 百度云下载 https://pan.baidu.com/s/1dFYaCM1 下载源码编译安装 2. 解压安装 tar -zxvf spark-2.2.0-bin-2.6.0-cdh5.7.0.tgz -C ~/app/ 3. 配置环境变量 vi .bashrc export SPARK_HOME=/home/ricky/app/spark-2.2.0-bin-2.6.0-cdh5.7.0 export PATH=$PATH:$SPARK_HOME/bin 3. 启动 cd $SPARK_HOME bin/spark-shell --master local[2]
